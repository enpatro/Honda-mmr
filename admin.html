How to add admin and users

  1. Create first admin in Firebase Authentication with email: e.patro@honda.hmsi.in
  2. Set password in Firebase Authentication only.
  3. Login here with first admin email.
  4. Create other users in Firebase Authentication.
  5. Ask each user to login once in the app so their profile appears below.
  6. Admin can set role as user or admin.

Important: Do not store real passwords inside GitHub files.
